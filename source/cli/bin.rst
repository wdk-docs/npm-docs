npm bin
============================================================================================

SYNOPSIS
-------------------

.. program:: npm

.. option:: bin

   Display npm bin folder

   .. code-block::

      npm bin [-g|--global]

DESCRIPTION
-------------------

Print the folder where npm will install executables.
