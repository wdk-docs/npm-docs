npm install-ci-test
============================================================================================

SYNOPSIS
-------------------

.. program:: npm

.. option:: install-ci-test

   Install a project with a clean slate and run tests

   .. code-block:: sh

      npm install-ci-test

.. option:: cit

   install-ci-test alias

DESCRIPTION
-------------------

This command runs an npm ci followed immediately by an npm test.

SEE ALSO
-------------------

- :option:`npm ci`
- :option:`npm test`
