npm-test
============================================================================================

Test a package

SYNOPSIS
-------------------

.. program:: npm

.. option:: test

  npm test [-- <args>]

  aliases: t, tst

DESCRIPTION
-------------------

This runs a package’s “test” script, if one was provided.
