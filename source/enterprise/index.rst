npm Enterprise
=============================

npm Enterprise is your companyâ€™s own npm registry, hosted on npm servers. npm Enterprise provides:

- Easy code discovery and sharing within your company
- Secure package deployment
- Unlimited Orgs and scopes
- Delegated authentication
- Access to the same packages available in the public registry
- Audit reports on potential security vulnerabilities in package dependencies

.. note:: These articles apply to npm Enterprise admins. End users of npm Enterprise should check out the rest of our documentation, particularly the "Getting started" and "Packages and modules" sections.
