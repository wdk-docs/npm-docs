关于npm CLI版本¶

npm命令行界面(CLI)以常规节奏发布。我们建议安装支持您的工作流程的版本:

  • 最新发布:最新的稳定版本。

  • 下一个版本:最新发布的npm版本，最终作为最新版本发布。

npm的最新版本¶

npm的最新版本是最新的稳定版本。安装Node.js时，会自动安装npm。但是，npm比Node.js更频繁地发布，因此要在命令行上安装最新的稳定版npm，请运行:

npm install npm@latest -g

npm的下一个版本¶

注解

npm的下一个版本可能包含与最新稳定版npm中最终发布的功能不匹配的功能。

npm的下一个版本是npm的最新未发布版本，最终作为最新版本发布。您可能希望将npm客户端更新到下一个版本，以便在最新版本发布之前针对它测试您的软件包。

要更新到npm的下一个版本，请在命令行上运行:

npm install npm@next -g

注解

根据开发周期，npm install npm@next -g 可能会重新安装最新版本的npm。
