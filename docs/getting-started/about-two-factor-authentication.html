关于双因素身份验证¶

双因素身份验证(2FA)通过使用确认您的身份来防止未经授权访问您的帐户:

  • 你知道的东西(比如你的用户名和密码)

  • 你有的东西(如手机或平板电脑)

当您启用2FA时，我们会在您对您的帐户或您具有写入权限的软件包执行某些操作时提示您输入唯一的一次性密码，具体取决于您的2FA配置。

注解

双因素身份验证可为您的帐户提供最佳的安全保护，以抵御攻击者。我们强烈建议您在注册后尽快在帐户中启用2FA。

npm上的双因素身份验证模式¶

npm上的双因素身份验证只能用于授权，或授权和写入。

仅限授权¶

如果您仅启用2FA授权，我们将为某些授权操作请求一次性密码。

行动

CLI命令

登录npm

npm login

更改个人资料设置(包括您的密码)

npm profile set

更改您的用户帐户的2FA模式

npm profile enable-2fa auth-only

为您的用户帐户停用2FA

npm profile disable-2fa

授权和写作¶

如果您启用2FA进行授权和写入，我们将为某些授权操作请求一次性密码，以及写入操作。

行动

CLI命令

登录npm

npm login

更改个人资料设置(包括您的密码)

npm profile set

更改您的用户帐户的2FA模式

npm profile enable-2fa auth-only

为您的用户帐户停用2FA

npm profile disable-2fa

创建令牌

npm token create

撤销令牌

npm token revoke

发布包

npm publish

取消发布包

npm unpublish

弃用包

npm deprecate

更改包可见性

npm access public/restricted

更改用户和团队包访问权限

npm access grant/revoke
