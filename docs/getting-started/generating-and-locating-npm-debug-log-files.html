Generating and locating npm debug.log files¶

当程序包无法安装或发布时，npm CLI将生成npm-debug.log文件。这个日志文件可以帮助你(和npm支持)弄清楚出了什么问题。

如果需要生成npm-debug.log文件，则可以运行其中一个命令。如果需要生成npm-debug.log文件，则可以运行其中一个命令。

用于安装包:

npm install --timing

对于发布包:

npm publish --timing

您可以在.npm目录中找到npm-debug.log文件。要查找.npm目录，请使用npm config get cache。

如果您使用 CI 环境，则您的日志可能位于其他位置。例如，在 Travis CI 中，您可以在 /home/travis/build 目录中找到它们。

注解

npm企业用户:如果您需要通过 enterprise@npmjs.com 联系npm企业支持，我们建议您附上 npm-debug.log 文件的全部内容或将内容复制到邮件正文中，以便我们更轻松诊断问题。
