在公共注册表上创建新用户帐户¶

如果您还没有npm用户帐户，则可以创建一个帐户，以便在公共注册表上共享和下载Javascript包。

注解

如果您使用的是npm Enterprise注册表，则必须使用SSO凭据登录，而不是创建帐户。有关更多信息，请与您公司的企业管理员联系。

在网站上创建一个帐户¶

  1. 转到npm注册页面。

  2. 在用户注册表单中，键入以下字段:

    • 全名

    • 公共电子邮件:您的公共电子邮件地址将添加到您的软件包的元数据中，并且任何下载软件包的人都可以看到。我们还会在您更新软件包时向此帐户发送电子邮件，以及不定期的产品更新和信息。

    • 用户名:在npmjs.com上发布包或与其他npm用户交互时将显示的用户名。您的用户名必须是小写，并且可以包含连字符和数字。

    • 密码:您的密码必须符合我们的密码指南。新帐户注册表单

  3. To sign up for our email newsletter, select “Sign up for the npm Weekly”. sign up for the npm weekly checkbox selected

  4. Select “Agree to the End User License Agreement and the Privacy Policy”. agree to the end user license agreement and privacy policy checkbox selected

  5. Click Create An Account. create an account button

注解

After signing up for an npm account, you will receive an account verification email. You must verify your email address in order to publish packages to the registry.

Testing your new account with npm login

使用npm login命令测试登录新帐户的情况。¶

注解

如果在使用npm login命令登录时拼错现有帐户用户名，则将创建一个包含拼写错误名称的新帐户。有关意外创建帐户的帮助，请联系npm支持。

  1. 在命令行上，键入以下命令，替换

    npm login

  2. When prompted, enter your username, password, and email address.

  3. If you have two-factor authentication enabled, when prompted, enter a one-time password.

  4. To test that you have successfully logged in, type:

    npm whoami

    Your npm username should be displayed.
