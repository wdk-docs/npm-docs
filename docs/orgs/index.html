Orgs¶

npm Orgs允许贡献者团队读取，编写和发布公共或私有范围包，以及由Org管理的未范围公共包。

公共包裹免费提供，私人包裹每个会员每月花费7美元。有关Orgs定价的更多信息，请参阅我们的定价页面。
