关于私人包¶

To use private packages, you must:

- be using npm version 2.7.0 or greater. To upgrade, on the command line, run npm install npm@latest -g
- have a paid user or Org account

With npm private packages, you can use the npm registry to host code that is only visible to you and chosen collaborators, allowing you to manage and use private code alongside public code in your projects.

私有包始终具有范围，默认情况下范围包是私有的。

  • User-scoped private packages can only be accessed by you and collaborators to whom you have granted read or read/write access. For more information, see 将协作者添加到用户帐户拥有的私有包.

  • Org-scoped private packages can only be accessed by teams that have been granted read or read/write access. For more information, see ../org/managing-team-access-to-org-packages.
