npm 企业版¶

npm Enterprise是您公司自己的npm注册表，托管在npm服务器上。 npm Enterprise提供：

  • 在您的公司内轻松发现和共享代码

  • 安全包部署

  • 无限的组织和范围

  • 委托身份验证

  • 访问公共注册表中可用的相同包

  • 审计报告包依赖关系中的潜在安全漏洞

注解

这些文章适用于npm Enterprise管理员。 npm Enterprise的最终用户应该查看我们的其他文档，特别是“入门”和“包和模块”部分。
