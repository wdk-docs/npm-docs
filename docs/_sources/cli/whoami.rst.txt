npm-whoami
============================================================================================


SYNOPSIS
-------------------

.. program:: npm

.. option:: whoami

   Display npm username

   .. code-block:: sh

      npm whoami [--registry <registry>]

DESCRIPTION
-------------------

Print the username config to standard output.
