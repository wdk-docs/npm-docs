npm stars
============================================================================================

SYNOPSIS
-------------------

.. program:: npm

.. option:: stars

   View packages marked as favorites

   .. code-block:: sh

      npm stars [<user>]

DESCRIPTION
-------------------

If you have starred a lot of neat things and want to find them again quickly this command lets you do just that.
