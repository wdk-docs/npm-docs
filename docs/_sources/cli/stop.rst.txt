npm stop
============================================================================================

SYNOPSIS
-------------------

.. program:: npm

.. option:: stop

   Stop a package

   .. code-block:: sh

      npm stop [-- <args>]

DESCRIPTION
-------------------

This runs a package’s “stop” script, if one was provided.
